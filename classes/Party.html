Static fundNewAccount

  • fundNewAccount(feePayerKey: PrivateKey, initialBalance?: { initialBalance: undefined | string | number | UInt64; isSameAsFeePayer: undefined | boolean | Bool }): void
    • Defined in lib/party.ts:715

    Use this method to pay the account creation fee for another account. Beware that you don't need to pass in the new account! Instead, the protocol will automatically identify accounts in your transaction that need funding.

    If you provide an optional initialBalance, this will be subtracted from the fee-paying account as well, but you have to separately ensure that it's added to the new account's balance.

    Parameters

    • feePayerKey: PrivateKey

      the private key of the account that pays the fee

    • initialBalance: { initialBalance: undefined | string | number | UInt64; isSameAsFeePayer: undefined | boolean | Bool } = {}

      the initial balance of the new account (default: 0)

      • initialBalance: undefined | string | number | UInt64
      • isSameAsFeePayer: undefined | boolean | Bool
